# Python Nonograms done with CMU-Graphics

## Note: This repository has been replaced with https://github.com/MSSTEM465/GodotBoardGames/

Keybinds:
  - Play Mode:
    - C - Config
    - J - Open JSON file
  - Create Mode:
    - G - Create Board
    - E - Export
  - Solve:
    - G - Start Manual Solve
    - R - Generate Manual Board Numbers
    - B - Solve using Manual Board Numbers
    - J - Solve using JSON
  - Arrow keys to move

  For me:
    - To build:
      - python -m PyInstaller --onefile nonogram.py
